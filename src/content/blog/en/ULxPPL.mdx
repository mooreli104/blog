I was getting bored and tired of doing a 6 day workout split (PPLxPPL, so I've decided to switch it up)


## The Workout Split

The workout is a 5-day/week ULxPPL split which means <Notation type="underline" color="yellow"> upper - lower - rest - push - pull - legs - rest.</Notation>
Hopefully this split helps me get bigger arms :P
<br/>


<Notation type="bracket" color="yellow" padding="10px" strokeWidth="2">
Day 1: Upper
-Incline bench 3x6-8
-Seated cable fly 2x12-15
-Weighted pull-up 3x6-8
-Cable lat raise 2x8-10
-Deficit pendlay row 2x6-8
-Overhead tricep ext. 2x8-10
-Bayesian bicep curl 2x8-10
-Cardio 20-30min
<br/>

Day 2: Lower
-Lying/seated leg curls 2x6-8
-Hack squats 3x6-8
-RDL 3x6-8
-Leg extension 2x8-10
-Hib Abduction 2x8-10
-Cable crunch 3x8-10
-Face pull 3x15-20
-Cardio 20-30min
<br/>

Day 3: Push
- Incline bench press 3x6-8
- Shoulder press 3x10-12
- Tricep extension 3x10-12
- Chest fly 3x10-12
- Lat raise 3x20
-Front raise 2x15-20
-Diamond push up 1xfailure
-Cardio 20-30min
<br/>

Day 4: Pull
- Mixed lat pulldown 3x12-15
- Pull-up 1xAMRAP
- Seated row 3x10-12
- Shrug 3x10-12
- Reverse fly 3x10-12
- Bicep curl 3x10-12
- Sulek curl 3x20
- Cardio 20-30min

<br/>
Day 5: Legs
- Deadlift 4x5
- Hipthrust 2x6-8
- Leg press 4x10-12
- Leg extension 3x8-10
- Seated calf raise 4x15-20
- Leg raise 3x8-10
- Decline sit-up 3x8-10
- Cardio 20-30min

</Notation>
